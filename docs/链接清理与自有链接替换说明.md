# LT导航链接清理与自有链接替换说明

## 本次处理目标

本次清理的目标是把仓库中原项目遗留的返佣链接、邀请码、优惠码、短链和个人导流入口替换为更适合公开导航站使用的内容，避免继续给原作者导流或产生返佣。

由于大多数平台的专属邀请链接必须先由账号本人在平台内申请或生成，因此本次**没有伪造或臆造你的专属链接**。当前处理方式如下：

- 能确认官方站点的，统一改为官方公开入口。
- 明显属于个人服务、资源中转、非官方分销或版权状态不清晰的，先从前台隐藏。
- D1 初始化种子与前台 fallback 数据同步更新，避免重建数据库后旧推广链接再次出现。

## 后续如何换成你自己的专属链接

### 方案一：直接在管理后台改

适合日常运营，推荐优先使用。

步骤：

1. 访问 `https://liantang.fun/admin`
2. 登录后台
3. 找到对应条目
4. 修改名称、链接、描述、标签
5. 保存后立即生效

适合后续替换为你自己的：

- 邀请链接
- 推荐码
- 联盟返佣链接
- 优惠码活动页

### 方案二：改仓库种子数据后重建

适合大批量改动或准备迁移到新数据库时使用。

需要同步修改：

- [src/lib/sites.json](/Users/summer/github/ref-baily-life/src/lib/sites.json)
- [migrations/0002_seed.sql](/Users/summer/github/ref-baily-life/migrations/0002_seed.sql)

如果只改后台而不改种子文件，平时访问没问题；但将来重建 D1 时，数据库初始内容不会自动带上后台后续新增的专属链接。

## 哪些平台通常可以申请你自己的专属链接

一般可自行申请的类型包括：

- 返现平台：TopCashback、Rakuten
- 银行与开户奖励：SoFi、Capital One、Varo、ZA 等
- 信用卡推荐：Chase、Amex
- 券商与交易所：Kraken、IBKR、Bybit、Bitget、Binance、OKX 等
- 支付与汇款：Wise、Revolut、LemFi
- 钱包与工具：OneKey、SafePal、Bitget Wallet、MiniMax 等
- eSIM 与服务：Tello、Xesim、RedteaGO 等

实际是否开放邀请资格，以平台当下规则为准。

## 运营建议

- 导航站默认优先放官方入口，稳定性最好。
- 你拿到自己的专属链接后，再逐条替换为你的链接。
- 前台最好长期保留一条说明：`部分链接可能包含合作或推广关系，请以页面说明为准。`
- 对涉及金融、开户、加密交易的平台，建议在描述里保留地区和合规提示，不要只写收益或奖励。
